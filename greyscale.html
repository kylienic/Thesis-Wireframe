Light intensity ranges from 0-255

0 (the absense of light) is the darkest possible value. 255 is the maximum amount of light. In a greyscale example, 0 is black, 255 is white, and the numbers in between are varying shades of grey.

Intensity values range between 0 and 255 for a total of 256 possible values. Why 256? Pixel information is stored in bits. 8 bits = 1 byte. One byte can represent 2^8=256 options. If we allot 1 byte to red, green and blue we end up with 3 byes or 24-bits of storage. This is what is referred to as 24-bit color.

If we allot a value in the 0-255 range to each of the red, green and blue values we end up with an RGB triple. If we want to use RGB triples to describe color, we can simulate the grey scale above by making each of the red, green and blue components identical. For example, (0,0,0) is black and (255,255,255) is white. The RGB grey scale is below.
