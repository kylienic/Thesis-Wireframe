What is Digital Imaging?

Digital Imaging encompasses the capture, processing and display of digital images and their data. It is an interdisciplinary field encompassing computer science, mathematics, photography, optics, and electrical engineering.







So... Let's Get Started!








What is a Digital Image?

A Digital Image is an Array of Pixels.











What is An Array?

An array is a 2-dimensional storage system. It has both width and height and holds values.



Image Pixels

Each pixel has an (x,y) location (width, height) as well as an RGB value (r,g,b).
