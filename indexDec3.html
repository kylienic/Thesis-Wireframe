What is a Digital Image?

A digital image is an expressed array of numeric values that represent light intensity. Yes folks, that's right: the images on your phone, your tumblr and your instagram are rows and rows of numbers. Pretty cool, eh?

Let's see it!
























A Digital Image is an array of pixels.

As you can see in this animation, an image is made up of rows and rows of pixels. In a color image, each pixel has three values: red, green, and blue (which you can see represented below). More on color later. For now, just focus on the concepts of pixels and arrays, which we will explore further.


Onward










An array is a grid of stored values.

Think of it like cubbies. An array is a 2-dimensional storage system, meaning that it has both rows and columns. Each square holds a pixel value.



Each Pixel has a location and an RGB value.

The RGB value (red, green, blue) will become important very soon. It is an a measure of light intensity.

Each pixel value can range from 0-255

0 (the absense of light) is the darkest possible value. 255 is the maximum amount of light. In a greyscale example, 0 is black, 255 is white, and the numbers in between are varying shades of grey.




Things get a little more complicated when we add in color. The range of values stays the same, but now you have three values instead of just one.
